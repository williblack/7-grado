¿Qué aprenderemos hoy?

Entenderemos cómo conectar una Base de Datos (una tabla de Excel) con una Carta para generar invitaciones o diplomas automáticamente sin tener que escribir uno por uno.

¿Cómo funciona la magia? ✨

📄

1. El Documento Maestro

(Pasa el ratón para ver)

Es la carta, diploma o invitación original que tiene espacios vacíos para los nombres.

📊

2. La Base de Datos

(Pasa el ratón para ver)

Es una tabla en Excel donde tenemos la lista de todos los nombres y datos que queremos usar.

🪄

3. La Fusión

(Pasa el ratón para ver)

La computadora une los dos archivos y ¡BOOM! crea cientos de cartas personalizadas en un segundo.

🤖 Actividad: El Laboratorio de Datos IA

PASO 1: En el Cuaderno ✍️

Dibuja una tabla vacía con 4 columnas y 4 filas. Las columnas serán: Nombre, Personaje Favorito, Poder (1-100) y Color.

PASO 2: Habla con la IA 🗣️

Copia este "Hechizo" (Prompt) y pégalo en el Chatbot (Gemini o ChatGPT) en la sala de sistemas:

"Hola IA. Crea una tabla con 5 personajes de videojuegos. Las columnas deben ser: Nombre, Personaje, Poder (número del 1 al 100) y Color. Dame la tabla lista para copiar a Excel."

Visualización de Datos (Ejemplo)

⚠️ Misión Final en la Sala de Sistemas:

  • Abre Microsoft Excel.
  • Copia la tabla que te dio la IA y pégala en la hoja.
  • Selecciona los datos y ponles Bordes y Colores.
  • Guarda el archivo como: Mis_Datos_IA.xlsx
